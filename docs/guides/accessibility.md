# Accessibility Guidelines

Ensuring Mackkas is usable by everyone.

## Standards
- **ARIA Roles:** Use appropriate roles for interactive elements (e.g., `role="button"`, `aria-label`).
- **Contrast:** Maintain high contrast ratios for text visibility.
- **Keyboard Navigation:** Ensure all features are accessible via `Tab` and `Enter`.
- **Alt Text:** Every image must have descriptive `alt` text.

## Tools
- **Lighthouse:** Run regular accessibility audits.
- **WAVE:** Web Accessibility Evaluation Tool for manual checks.
