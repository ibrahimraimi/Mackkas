# SEO Optimization

Best practices implemented to improve search visibility.

## On-Page SEO
- **Semantic HTML:** Use of `<h1>` through `<h4>` tags for proper content hierarchy.
- **Alt Text:** Descriptive alt text for all product images.
- **Descriptive Titles:** Page-specific `<title>` tags.
- **Meta Descriptions:** Compelling summaries for search engine snippets.

## URL Structure
- **Kebab-case URLs:** Paths like `/product-catalog` instead of `/Mackkas.html`.
- **Asset Naming:** Images include keywords like `blue-suit` or `leather-handbag`.

## Future Enhancements
- **Dynamic Meta Tags:** Product-specific meta tags based on database content.
- **Sitemap Generator:** Automatic generation of `sitemap.xml`.
- **JSON-LD Schema:** Structured data for product Rich Results in Google Search.
