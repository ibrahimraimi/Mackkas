# Tech Stack

The Mackkas platform is built using a lightweight and efficient tech stack to ensure performance and maintainability.

## 🎨 Frontend
- **HTML5**: Semantic structure.
- **CSS3**: Modern, responsive styling with Vanilla CSS.
- **JavaScript (ES6+)**: Core business logic and DOM manipulation.
- **Iconify**: Scalable vector icons for the UI.

## ⚙️ Backend
- **Python**: Primary programming language.
- **Flask**: Micro-framework for web serving and API.
- **Werkzeug**: Utilities for hashing and security.

## 🗄️ Database
- **SQLite**: File-based database for simplicity and portability.
- **SQLAlchemy**: ORM for Pythonic database management.

## 🛠️ Tools
- **Pip**: Package management.
- **Mermaid**: For architecture and schema diagrams in documentation.
- **Jinja2**: Templating engine for dynamic HTML rendering.
