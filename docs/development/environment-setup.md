# Environment Setup

Guidelines for setting up a consistent development environment.

## 1. Required Hardware
- **Processor:** 1.6 GHz or faster.
- **RAM:** 4 GB or more.
- **Storage:** At least 500 MB of free space.

## 2. Software Stack
- **Python 3.8+**
- **Node.js (Optional):** Only if you need to run frontend build tools or linters.
- **Git**

## 3. Recommended Extensions (VS Code)
- **Python (Microsoft):** Linting, debugging, and IntelliSense.
- **SQLite Viewer:** To view the database directly in the IDE.
- **Prettier:** For consistent JS/CSS formatting.
- **ESLint:** For JavaScript linting.

## 4. Configuration
Ensure your terminal is using the correct virtual environment for all `pip` and `python` commands. Set up a `.env` file for local secrets (e.g., `SECRET_KEY`).
