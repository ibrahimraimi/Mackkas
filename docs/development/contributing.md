# Contributing Guide

We welcome contributions to the Mackkas platform! Please follow these guidelines:

## How to Contribute
1. **Fork the repo.**
2. **Create a branch:** `git checkout -b feature/your-feature-name`.
3. **Follow standards:** Review the [Coding Standards](../development/coding-standards.md) before writing code.
4. **Commit changes:** Use descriptive commit messages.
5. **Push and PR:** Push to your fork and submit a Pull Request to the `develop` branch.

## Review Process
- All PRs must pass basic linting and tests.
- Maintainers will review the code for security, performance, and adherence to the architecture.
- Address any requested changes promptly.

## Code of Conduct
Please be respectful and professional in all communications.
