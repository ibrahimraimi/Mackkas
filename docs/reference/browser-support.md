# Browser Support

Mackkas aims for broad compatibility with modern web browsers.

## Supported Browsers
| Browser | Minimum Version |
|---|---|
| Google Chrome | 90+ |
| Mozilla Firefox | 88+ |
| Safari | 14+ |
| Microsoft Edge | 90+ |

## Mobile Support
- iOS Safari (iOS 14+)
- Android Chrome (Latest)

## Non-Supported Browsers
- Internet Explorer (All versions)
- Legacy browsers without ES6+ support.
