# Reference Documentation

Technical details and terminology for the Mackkas platform.

## Glossary
- **ORM:** Object-Relational Mapping (SQLAlchemy).
- **Kebab-case:** A naming convention where words are lowercase and separated by hyphens (e.g., `this-is-kebab-case`).
- **Jinja2:** The templating engine used by Flask.
- **Scrypt:** The hashing algorithm used for passwords.

## Dependencies
Core dependencies used in the project:
- `flask`: The web framework.
- `flask-sqlalchemy`: Database integration.
- `werkzeug`: Security utilities.
- `pytest`: Testing framework.

## Browser Support
- Chrome (latest 2 versions)
- Firefox (latest 2 versions)
- Safari (latest 2 versions)
- Edge (latest 2 versions)
- Mobile Safari & Chrome for Android.
