# Dependencies

Detailed list of third-party packages and tools used in Mackkas.

## Python Packages
- `Flask`: Minimalist web framework.
- `Flask-SQLAlchemy`: Database toolkit.
- `Werkzeug`: WSGI utilities and security.
- `Pytest`: Robust testing framework.

## Frontend Libraries
- `Iconify`: Scalable icons.
- `Google Fonts`: Typography (Inter, Roboto).

## Infrastructure Tools
- `SQLite`: Relational database storage.
- `Gunicorn`: Production WSGI server (recommended).
- `Nginx`: Reverse proxy server (recommended).
