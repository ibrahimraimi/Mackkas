# Prerequisites

The Mackkas platform requires the following software to be installed:

## System Requirements
- **Operating System:** Linux, macOS, or Windows 10/11.
- **Python:** Version 3.8 or higher.
- **Web Browser:** Modern browser (Chrome, Firefox, Safari, Edge).

## Python Dependencies
The following core libraries are required (automatically installed via `requirements.txt`):
- `Flask`: Web framework.
- `Flask-SQLAlchemy`: ORM for database management.
- `Werkzeug`: Utility library for password hashing.

## Development Tools (Optional)
- `Git`: For version control.
- `VS Code`: Recommended IDE with Python extension.
- `DB Browser for SQLite`: For viewing the database directly.
