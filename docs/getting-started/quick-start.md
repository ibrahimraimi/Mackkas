# Quick Start

Get up and running with Mackkas in under 5 minutes.

## Prerequisites
Ensure you have Python 3.8+ installed on your system.

## One-Line Setup
If you are on a Unix-like system, you can run:
```bash
pip install flask flask-sqlalchemy werkzeug && python3 seed.py && python3 app.py
```

## Basic Usage
1. Open your browser to `http://127.0.0.1:5000`.
2. Navigate to the **Signup** page to create an account.
3. **Login** with your new credentials.
4. Browse the **Mackkas** catalog and add items to your cart.
5. View your cart and proceed to checkout (simulated).

## Navigation
- `/`: Home Landing page
- `/mackkas`: Product catalog
- `/login`: User login
- `/signup`: User registration
- `/logout`: Clear session and logout
